## Notes On Licensing

### GPL Compatibility

All Licenses used in this Project are Compatible with each other & do not impose viral Copyleft Requirements on the Main PEND Boilerplate Codebase.

### LGPL Exception (`psycopg2`)

The `psycopg2` Library is Licensed under LGPL with a Special Exception that allows its use in Proprietary Applications when used as a Library (which is how it's used in this Project). This does not affect the MIT licensing of the PEND Boilerplate.

### Commercial Use

All Dependencies are safe for Commercial Use without Restrictions. The most Restrictive License is LGPL (`psycopg2`), which only requires that modifications to `psycopg2` itself be shared if distributed.

### Updating This Document

When Adding New Dependencies :

1. Identify the License Type
2. Add an Entry in the Appropriate Section
3. Include Version, Copyright Holder & License URL
4. Verify License Compatibility
